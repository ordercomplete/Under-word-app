Як це все працює:

1. Кнопка в App.js:
   З'являється тільки в режимі розробки (process.env.NODE_ENV === 'development')

Зберігає стан показу тестувальника в localStorage

Знаходиться у верхньому правому кутку

2. FormatTester компонент:
   Фіксована панель у правому нижньому кутку

Можна згортати/розгортати

Тестує:

Формат словників Strong

Формат перекладів

Роботу JSON адаптера

Доступність файлів

3. Утиліти тестування:
   Перевірка існування файлів

Аналіз формату JSON

Порівняння розмірів файлів

Генерація звітів

4. Як використовувати:
   Увімкнути тестувальник: Натиснути кнопку "JSON тест" у правому верхньому кутку

Запустити тести: Натиснути "Запустити тести" у тестувальнику

Перемкнути формат: Натиснути "Перемкнути формат" для зміни між скороченим/повним

Перезавантажити: Натиснути "Перезавантажити" після зміни формату

5. Що показує тестувальник:
   ✅ Успішні тести:

Словник завантажено з правильним форматом

Переклади доступні

Адаптер працює коректно

⚠️ Попередження:

Файли не знайдені в одному з форматів

Формат визначити не вдалося

❌ Помилки:

Файли недоступні

Неправильний JSON формат

Проблеми з адаптацією

6. Для продакшену:
   Просто видаліть або закоментуйте частини коду, пов'язані з FormatTester, або оберніть їх умовою:

javascript
{process.env.NODE_ENV === 'development' && (
// Весь код тестувальника
)}
