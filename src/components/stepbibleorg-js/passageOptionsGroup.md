**Створення React-компонента на основі HTML і CSS**
Так, ви можете створити React-компонент на основі наданого HTML-коду панелі passageOptionsGroup та CSS-стилів, адаптувавши їх до JSX-синтаксису. Я також оновлю посилання для поширення в соціальних мережах, щоб вони відповідали актуальному контексту (наприклад, поточна дата та правильні URL). Нижче наведено покроковий процес і код.

# Крок 1: Адаптація HTML до React (JSX)

Замінимо class на className.
Замінимо onclick на onClick і додамо обробники подій (якщо потрібно).
Використаємо React-компоненти для динамічних частин (наприклад, кнопок і випадаючих меню).
iframe для соціальних мереж залишимо як є, але оновимо атрибути.

Замінимо застарілі параметри (наприклад, time=1760798099030) на динамічні значення.

# Крок 3: Інтеграція CSS

Скопіюємо стилі в окремий CSS-файл (наприклад, PassageOptionsGroup.css) і імпортуємо його в компонент.
Додамо префікс .PassageOptionsGroup для уникнення конфліктів стилів.

**Пояснення**

React-компонент:

Компонент PassageOptionsGroup є функціональним і використовує useState для управління станом (поточний уривок і версії).
Обробники подій (handleSelectVersions, тощо) замінюють step.util і є заглушками. Їх можна розширити за потребою.

Оновлені посилання для соціальних мереж:

Twitter: Оновлено src з динамічним time і коректним url.
Facebook: Оновлено href і src iframe з актуальним URL.
Текст і URL енкодуються для коректної передачі.

Стили:

Додано префікс .PassageOptionsGroup до всіх селекторів, щоб стилі застосовувалися тільки до цього компонента.
Збережено всі оригінальні стилі, адаптувавши їх до React-структури.

Інструкції для використання

Збережіть PassageOptionsGroup.js і PassageOptionsGroup.css у відповідних папках (наприклад, src/components).
Імпортуйте компонент у ваш додаток: import PassageOptionsGroup from './components/PassageOptionsGroup';.
Переконайтеся, що у вашому проєкті підключено Bootstrap (для glyphicon) і змінні --clrText, --clrBackground, --clr2ndHover визначені в глобальних стилях.

Тестування

Запустіть npm start або tauri dev.
